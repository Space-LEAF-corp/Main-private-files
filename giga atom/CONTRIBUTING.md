# Contributing to the Giga Atom Project
This repository is maintained by Space Leaf Corp under the stewardship of Leif William Sogge.  
All contributions must align with the mission of planetary benefit, safety, and non‑weaponized research.

## 1. Eligibility
Only approved collaborators with explicit written authorization from Space Leaf Corp may contribute.

## 2. Contribution Principles
- All contributions must support **nonprofit, planetary-benefit research**.
- No code may introduce or enable **weaponization**, harm, or misuse.
- All contributions must comply with the **Space Leaf Corp Proprietary License 1.0**.
- Contributors must follow the **Code of Conduct** included in this repository.

## 3. Workflow
1. Create a feature branch from `main`.
2. Submit a Pull Request (PR) with:
   - A clear description of the change.
   - Justification for how it supports safety and planetary benefit.
   - Tests for any new functionality.
3. PRs require:
   - At least one reviewer approved by Space Leaf Corp.
   - Passing CI checks.
   - No security regressions.

## 4. Prohibited Contributions
- Any feature that could be repurposed for harm.
- Any attempt to bypass safety, stability, or ethical constraints.
- Any code that violates international law or ethical research standards.

## 5. Ownership
All contributions become the property of **Space Leaf Corp** and fall under the proprietary license.

Thank you for supporting safe, ethical, planetary‑aligned research.
