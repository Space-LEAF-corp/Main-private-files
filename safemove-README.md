# SafeMove: Digital Body‑Safety Detection Engine  
### Behavior‑Based Protection for Kids, Teens, and Adults

SafeMove is a behavior‑based safety system designed to detect **unsafe physical‑movement requests**, **camera‑angle manipulation**, and **boundary‑testing behavior** in online messages.

It is:
- **Kid‑safe**
- **Gender‑neutral**
- **Non‑graphic**
- **Behavior‑focused**
- **Platform‑agnostic**
- **Suitable for professional deployment**

SafeMove helps protect users — especially minors — from manipulative or exploitative online interactions by identifying red‑flag phrases and returning a clear risk assessment.

---

## ✨ Features

### ✔ Behavior‑Based Detection  
Identifies unsafe requests such as:
- “Bounce up and down”
- “Do jumping jacks”
- “Turn around”
- “Move the camera down/up”
- “Show me from another angle”
- “Do that again but slower/faster”

### ✔ Risk Scoring  
Returns:
- **low** — no red flags detected  
- **medium** — one red flag detected  
- **high** — multiple red flags detected  

### ✔ JSON API (FastAPI or Flask)  
Platforms can POST a message and receive:
- risk level  
- matched phrases  
- matched categories  
- explanation  

### ✔ Educational Content Included  
The repository includes:
- Kid‑Safe Handbook  
- Parent Training Guide  
- School Curriculum Packet  
- Platform Policy Recommendations  

These can be used in apps, dashboards, or documentation.
