# HTML Seven — Emoji Texture Engine

HTML Seven is a modular, emoji‑driven visual engine that generates animated
wallpapers, interactive emoji tiles, and optional audio‑emitting nodes.  
Modes are defined in JSON and loaded dynamically, allowing creators to build
custom emoji “textures,” animations, and sound behaviors without editing code.

---

## 🚀 Features

- Emoji‑based animated wallpaper generator  
- Mode system powered by JSON (Calm, Party, Cosmic, Neon Drift, etc.)  
- Audio nodes (local files or streaming URLs)  
- Modular architecture (`engine.js`, `animation.js`, `audio.js`, `renderer.js`)  
- Easy to extend with new modes, animations, or effects
