# GQI&EA — General Question Inquiry & Engine Analysis

GQI&EA is a lightweight, public-friendly inquiry tag system designed to let users
submit questions with automatic tagging, timestamping, and routing.

Each inquiry is automatically stamped with:
- Username
- Timestamp
- User’s unique string tag

The engine then performs a quick analysis to route the inquiry to the correct
module, team, or subsystem.

This repo contains:
- Documentation for public and internal use
- Core logic for tag generation and timestamping
- Routing engine stubs
- Example usage patterns
