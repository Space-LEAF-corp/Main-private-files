## v2.0 — Advanced Linguistic Engine (Current)
**Release Date:** TBD  
**Status:** Active Development

### New Features
- Dialect Recognition Layer (AAVE, Southern, Caribbean, UK, AU, etc.)
- Phonetic Reconstruction Engine for accent-driven pronunciation
- Multilingual Context Engine (code-switching support)
- Tone & Identity Lock (expanded)
- Refinement Modes:
  - Preserve Dialect
  - Light Refinement
  - Full Refinement
  - Dialect → Standard English
- Pronunciation Learning Loop
- Expanded test suite for dialects and multilingual speech

### Improvements
- Stronger noise immunity
- Better slang preservation
- More transparent correction diffs
